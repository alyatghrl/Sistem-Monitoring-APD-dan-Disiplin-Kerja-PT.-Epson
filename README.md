# Sistem-Monitoring-APD-dan-Disiplin-Kerja-PT.-Epson
Proyek ini adalah sistem pemantauan Kesehatan dan Keselamatan Kerja (K3) berbasis Computer Vision. Sistem dirancang untuk mendeteksi secara otomatis kepatuhan penggunaan Alat Pelindung Diri (APD) pada lingkungan kerja industri. Fokus utama deteksi meliputi Helm Keselamatan (Safety Helmet) dan Rompi Kerja (Safety Vest).
